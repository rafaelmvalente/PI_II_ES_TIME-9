O NotaDez é um sistema web desenvolvido para docentes do ensino superior gerenciarem as notas de seus estudantes de forma independente dos sistemas institucionais. A aplicação permite o controle completo do processo avaliativo, desde o cadastro de instituições e disciplinas até o lançamento de notas e cálculo de médias finais.

Funcionalidades:
- Cadastrar o docente
- Cadastrar disciplinas
- Cadastrar turmas, matérias e alunos
- Lançar notas
- Exportar dados
- Alterar perfil

Tecnologias Utilizadas:

- Node.js
- TypeScript e JavaScript
- Express.js
